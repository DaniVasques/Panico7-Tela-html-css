# 🎬 Pânico 7

Landing page desenvolvida em *HTML* e *CSS, inspirada no filme **Pânico 7*.  
O projeto apresenta um layout temático com pôster, sinopse, classificação indicativa e botão interativo com acesso ao trailer.

------------------------------------------------------------------------------------------------------------------------------------------------------------------

## 🚀 Tecnologias
- HTML5
- CSS3
- Google Fonts

## ✨ Recursos
- Layout temático de filme
- Botão com efeito hover
- Link externo para trailer
- Responsividade básica

## 📚 Aprendizados
Este projeto foi desenvolvido como atividade prática das minhas primeiras aulas de Front-End, com foco em estruturação de páginas e estilização visual.

## ▶️ Como visualizar
Basta abrir o arquivo index.html no navegador.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------

## Objetivo do projeto

Este projeto foi criado para praticar os fundamentos de HTML e CSS, com foco na construção de uma landing page temática, organização de conteúdo e estilização visual inspirada no universo de suspense do filme Pânico 7.
